# Simulador de Investimentos

Planilha automatizada para simulação de investimentos com cálculo de patrimônio acumulado e dividendos mensais.

## Objetivo
Permitir que o usuário simule cenários de investimento considerando:
- Valor inicial
- Aportes mensais
- Taxa de rendimento
- Período de investimento

## Funcionalidades
- Cálculo automático do valor total investido
- Projeção de patrimônio acumulado
- Estimativa de dividendos mensais
- Simulação de diferentes cenários
